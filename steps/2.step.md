
# Chapter: Creating First API & Testing with Postman

---

## Section: Creating First API (Django REST Framework)

1. **Install Django REST Framework**

   * To build APIs in Django, install Django REST framework:

     ```bash
     pip install djangorestframework
     ```

2. **Add REST Framework to Django Settings**

   * Open `settings.py` inside the inner `backend` folder.
   * Add `'rest_framework'` to the `INSTALLED_APPS` list.

3. **Create `views.py` File**

   * Inside the inner backend folder (`backend/backend/`), create a new file named `views.py`.

4. **Add First API Code**

   * In `views.py`, add the following code:

     ```python
     from rest_framework.decorators import api_view
     from rest_framework.response import Response

     @api_view(['GET'])
     def hello_api(request):
         return Response({"message": "Hello from Django API!"})
     ```

5. **Configure URLs (Inner Backend Folder)**

   * Open `backend/backend/urls.py`.
   * Add the API route:

     ```python
     from django.contrib import admin
     from django.urls import path
     from .views import hello_api

     urlpatterns = [
         path('admin/', admin.site.urls),
         path('hello/', hello_api, name='hello_api'),
     ]
     ```

6. **Configure URLs (Root-Level Backend Folder)**

   * Open the root-level `backend/urls.py`.
   * Include the inner backend URLs:

     ```python
     from django.contrib import admin
     from django.urls import path, include

     urlpatterns = [
         path('admin/', admin.site.urls),
         path('', include('backend.urls')),
     ]
     ```

7. **Run the Django Server**

   * Start the backend server:

     ```bash
     python manage.py runserver
     ```

8. **Test API in Browser**

   * Open the browser and visit:

     ```
     http://localhost:8000/hello/
     ```
   * You should see the Django REST framework page with:

     ```
     Hello from Django API!
     ```

## Section: Testing the API Using Postman

1. **Install Postman**

   * Download and install the Postman application.

2. **Create and Log in to Postman Account**

   * Open Postman.
   * Create an account and log in.

3. **Create a Workspace**

   * Create a new workspace.
   * Name it **Job Portal** (or **React-Django**).

4. **Create a Collection**

   * Inside the workspace, create a new collection.
   * Name it **Test**.

5. **Create a Request**

   * Inside the **Test** collection, add a new request.
   * Name the request **Hello API**.

6. **Configure the Request**

   * Set the request method to **GET**.
   * Copy the API URL from the browser:

     ```
     http://localhost:8000/hello/
     ```
   * Paste it into the URL field.

7. **Send the Request**

   * Click the **Send** button.

8. **Verify the Response**

   * You should receive the JSON response:

     ```json
     {
       "message": "Hello from Django API!"
     }
     ```
